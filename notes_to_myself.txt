

This app was based on the series of tutorials by SuperCoder ("Python Django Student Management System, https://youtu.be/y3llbdTtam4). One version can be found on GitHub: https://github.com/hackstarsj


The version for testing deployment on Heroku is called "Student Management System".

To use Heroku:

1. Add file Procfile
2. Add wsgi stuff to Profile: web: 
gunicorn student_management_system.wsgi --log-file -
3. Add file runtime.txt
4. Add Python version to runtime.txt
5. $ heroku loginyu
6. pip3 install gunicorn dj-database-url whitenoise 
7. brew install postgresql (BEFORE pip3 install psycopg2-binary)
8. pip3 install psycopg2-binary
9. Add to settings.py: STATICFILES_STORAGE='whitenoise.django.GzipManifestStaticFilesStorage'
10. Add to settings.py Middleware: 
    whitenoise.middleware.WhiteNoiseMiddleware
    import dj_database_url
    prod_db=dj_database_url.config(conn_max_ag=500)
    DATABASES['default'].update(prod_db) 
11. $ heroku create student-manage-csp
    (heroku apps:rename newname --app oldname)
12. ALLOWED_HOSTS = ["student-manage-csp.herokuapp.com"]
13. $ heroku git:remote student-manage-csp
14. Add .keep (an empty file) to (the empty) static folder
    (or add a file)
    (or: $ heroku config:set DISABLE_COLLECTSTATIC=1 )
14. git push heroku master
15. $ heroku run python manage.py makemigrations
    $ heroku run python manage.py migrate

15. $ heroku open
